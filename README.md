# Zeev Dashboard

Painel de indicadores conectado ao Supabase.

## Deploy na Vercel (mais rápido)

1. Crie um repositório no GitHub e faça push deste projeto
2. Acesse [vercel.com](https://vercel.com), conecte o repositório
3. Clique em **Deploy** — a Vercel detecta Vite automaticamente
4. Pronto! O dashboard estará online em ~1 minuto

## Rodar localmente

```bash
npm install
npm run dev
```

Acesse `http://localhost:5173`
